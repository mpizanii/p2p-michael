# Documentação do Projeto P2P

Documentação técnica e acadêmica do sistema distribuído P2P desenvolvido para a disciplina de
Arquitetura de Sistemas Distribuídos (ADS — CEUBr 2026).

## Índice

| Documento                           | Conteúdo                                                                |
|-------------------------------------|-------------------------------------------------------------------------|
| [ARCHITECTURE.md](ARCHITECTURE.md)  | Visão geral, componentes, diagramas Mermaid, modelo de threads, histerese |
| [PROTOCOL.md](PROTOCOL.md)          | Todos os formatos de mensagem com exemplos JSON e diagramas de sequência |
| [SPRINTS.md](SPRINTS.md)            | Descrição detalhada de cada sprint e evolução das funcionalidades        |
| [DECISIONS.md](DECISIONS.md)        | Registros de Decisões Arquiteturais (ADR-001 a ADR-009)                  |
| [TESTING.md](TESTING.md)            | 60 casos de teste, cobertura por requisito, resultados                   |
| [EXECUTION.md](EXECUTION.md)        | Instruções passo a passo para cada cenário de demonstração               |

## Pasta superpowers/

Contém artefatos de design gerados com suporte de IA durante o processo de desenvolvimento:

| Pasta                   | Conteúdo                                                  |
|-------------------------|-----------------------------------------------------------|
| `superpowers/specs/`    | Especificações técnicas detalhadas (ex: design da eleição UDP) |
| `superpowers/plans/`    | Planos de implementação com checklist e estimativas       |

Esses documentos refletem o processo de design, não o estado final do código.
Para o estado atual, consulte os documentos acima.
